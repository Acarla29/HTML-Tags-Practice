HTML Tags

Headings

Headings come in different levels: h1 for the top-level heading, h2 for the top sub-heading, and so on

  • every page should have one, and only one h1 tag
  • headings from `h1` to `h6` identify blocks and sections

Paragraphs

Paragraphs of long-form text containing complete sentences go in the p (paragraph) tag.

Lists

  1. Ordered lists

    2. Ordered lists are numbered and go in `ol` tags. Each list item goes in an `li` tag.

  2. Unordered lists

    3. Unordered lists are marked with bullet points and go in `ul` tags. Each list item goes in an `li` tag.

Anchors (Links)

  • Now, you think you'd use the link element to do that. That's not what you use. Instead, you use an anchor element (a).

  • Anchor tags create links to other pages. The URL goes in the `href` attribute.
    • Open link to surch for more info

  • Setting `target="_blank"` will open the link in a new window.
    • Opens MDN

  • Can be used to navigate to a different location on the current HTML page.
    • Click to go to the top of the page

Text Emphasis

Italic

The `em` element adds extra emphasis to the text within the opening and closing tags.

Bold

You can also use the `strong` element this shows up as bold text.
